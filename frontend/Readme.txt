After downloading this file, please follow these steps
1.) Open any terminal window and run npm install to install all dependencies
2.) run -> npm run start (This will start the development server at port 3000)
3.) Open browser and view the project on http://localhost:3000


Points->
The entire design has been coded from scratch.
No frontend libraries like bootstrap, bulma, material design were used to make the code minimal.
Focus was given mainly for implementation of all the points given thoroughly.
React-toast-notifications library was used for displaying custom alert
Cart items state saving was done using localStorage.
